Usage of g3dogl: 

g3dogl.exe **.m 

1. left mouse: rotation 
right mouse: scaling 
middle mouse: translation 

2. D + e: visualize the triangulation 
D + s: show frame
D + r: change all face normal, so that inside surface 
rendered with green becomes outside

3. ii: (hide text)

4. r: updating 

5. E: eneter edit status
after coming to edit mode, put arrow on the target 
edge or face or vertex, you can do the following:

t: sharp edge 
s: edge split
w: edge swap
f: remove one face
v: remove one vertex

@@: save changes to the mesh file

3. g3dogl.exe **_uv.m -texturemap check.bmp
D + t, visualize the texture mapping.

g3dogl的用法:

g3dogl。exe * * 00

1。鼠标:旋转
鼠标右键:扩展
鼠标中:翻译

2。D + e:可视化三角
D + s:展示架
D + r:改变面对一切正常,内表面
呈现绿色变成了外面

3所示。2:(隐藏文本)

4所示。r:更新

5。艾凡:硅谷动力编辑状态
进入编辑模式后,给目标箭头
边或面或顶点,可以执行以下操作:

t:锋利的边缘
s:边缘分割
w:边交换
f:删除一个面
v:删除一个顶点

@@:保存修改网格文件

3所示。g3dogl。exe * * _uv。m -texturemap check.bmp
D + t,可视化纹理映射。
